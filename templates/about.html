About The Project

Project Overview

This project is an AI-Based Autism Behavioral Screening System developed using Machine Learning and Flask.

The system analyzes behavioral questionnaire responses and predicts autism traits using a trained Random Forest Machine Learning model.

Technologies Used

  • Python
  • Flask
  • Scikit-learn
  • SQLite
  • HTML/CSS
  • Machine Learning

Machine Learning Algorithm

Random Forest Classifier is used for prediction because of its high accuracy and ability to handle structured behavioral data efficiently.

Future Scope

  • Voice-based autism detection
  • Mobile application integration
  • Cloud deployment
  • Advanced AI analytics
