# AI-Based Autism Behavioral Screening System

## Project Overview

This project is an AI-powered web application developed using Machine Learning and Flask for behavioral autism screening.

The system analyzes behavioral questionnaire responses and predicts autism traits using a trained Random Forest Machine Learning model.

---

## Features

- AI-based autism prediction
- Behavioral questionnaire system
- Random Forest Machine Learning model
- Professional responsive UI
- SQLite database integration
- Prediction history management
- Analytics dashboard
- Donut chart visualization
- Modern healthcare-themed design

---

## Technologies Used

- Python
- Flask
- Scikit-learn
- SQLite
- HTML
- CSS
- Machine Learning
- Matplotlib

---

## Machine Learning Algorithm

The project uses:
### Random Forest Classifier

Reason for using:
- High accuracy
- Handles structured data efficiently
- Reduces overfitting
- Suitable for healthcare prediction systems
